# Python Modules and Packages Assignment

## 📌 Overview
This project demonstrates the creation and usage of Python modules and packages.

It focuses on:
- Creating custom Python modules
- Importing modules in different ways
- Creating packages using __init__.py
- Importing functions from packages
- Organizing a structured project folder

No OOP or file handling is used as per assignment restrictions.

## 🔹 Modules

### math_utils.py

- subtract(a, b)

### string_utils.py
- capitalize_words(text)
- reverse_string(text)
- word_count(text)
